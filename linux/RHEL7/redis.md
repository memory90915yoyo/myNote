# Redis

![redis](https://redis.io/images/redis-white.png)


---

## Install

```
# install package
centos:~ # yum install redis

# start service
centos:~ # systemctl start redis
centos:~ # systemctl enable redis

# test service
centos:~ # redis-cli ping
```

## Configure

```
centos:~ # vi /etc/redis.conf
```
